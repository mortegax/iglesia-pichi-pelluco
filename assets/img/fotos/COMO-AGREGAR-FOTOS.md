# Fotos del sitio

Las fotos reales de la iglesia ya están integradas. Estas son las que usa la web
(optimizadas desde la carpeta `Fotos/` de la raíz):

| Archivo | Dónde aparece |
|---|---|
| `hero.jpg` | Fondo sutil de la portada (Hero) — santuario completo |
| `historia.jpg` | Sección Historia, con la insignia "+50 años" |
| `congregacion.jpg` | Galería (destacada) — congregación en adoración |
| `grupo.jpg` | Galería (destacada) — comunión y servicio |
| `coro.jpg` | Galería — Coro Instrumental |
| `juventud.jpg` | Galería — Juventud Mixta (danza) |
| `palabra.jpg` | Galería — Escuela Dominical |
| `adoracion.jpg` | Galería — Culto General |

## Para cambiar una foto
1. Reemplaza el archivo `.jpg` correspondiente en esta carpeta (mismo nombre), **o**
2. Déjame una foto nueva en la carpeta `Fotos/` de la raíz y dime cuál reemplazar.

Tamaño ideal: ~1500 px de ancho, formato `.jpg`. Yo las optimizo con `sips`.
